# DocKey

## Description

DocKey est un prototype de site web en HTML/CSS pour créer, ouvrir et gérer des documents.

Le site contient plusieurs pages simples qui permettent de naviguer entre :
- page d'accueil avec accès à "Créer Doc" et "Ouvrir Doc"
- page de connexion
- page de création de compte
- page de liste des documents
- page d'éditeur de texte

## Structure du projet

- `index.html` : page d'accueil principale
- `page/Creer_doc.html` : page pour créer un document
- `page/Ouvrir_doc.html` : page pour ouvrir un document avec un code
- `page/page_connection.html` : page de connexion
- `page/page_créer_compte.html` : page d'inscription
- `page/page_liste.html` : page de liste des documents
- `page/editeur.html` : page d'édition de document
- `style/style.css` : styles partagés du site
- `source/LOGO.png` : logo utilisé sur la page d'accueil
